## What It Does

Lifts a passage verbatim from a source document and places it into the section with no modifications. The content appears exactly as it does in the source.

## How It Works

This action reads your source files, finds the relevant passage, and copies it directly into the section without any rewriting, summarizing, or modification.

**Chunk behavior.** Copy from Source copies one chunk of text at a time, where a chunk is the text that appears between two consecutive headings in a source document. If a source document has a heading "Adverse Events" followed by a block of text before the next heading, that block is one chunk. The action identifies which chunk is relevant to the section and copies it.

**No modifications.** Unlike Copy and Modify from Source, this action makes no changes to the copied content. Tense, terminology, and formatting are preserved as-is from the source.

**Source must contain the exact text you want.** Because the action copies verbatim, the quality of the output depends entirely on the quality of the source content. If the source text needs to be updated or adjusted for the target document, use Copy and Modify from Source instead.

## Example Instructions

| Weak instruction | Strong instruction |
| --- | --- |
| "Copy from the protocol" | "Copy the study objectives section verbatim from the protocol. Include primary and secondary objectives as written." |
| "Use the source text" | "Copy the statistical methods section from the SAP verbatim. Include the section on the primary analysis, including the statistical model and hypothesis test." |

## When to Use This Action

**Use Copy from Source when:**

- The source text should appear exactly as written
- No rewriting, summarizing, or modification is needed
- The section is drawing from a single, well-defined chunk of source content
- Verbatim accuracy is critical (e.g., regulatory submissions where exact language matters)

**Use a different action when:**

- The content needs to be modified (tense changes, terminology updates) → use **Copy and Modify from Source**
- The content needs to be condensed or synthesized → use **Summarize from Source**
- You're working from a prior version of the document → use **Copy and Update**

## Tips

- If the action is picking up the wrong chunk, check the heading structure in your source document. The action uses headings to define chunk boundaries, so unclear or missing headings can cause it to grab the wrong passage.
- For content that spans multiple chunks (multiple sections in the source), you may need to use multiple Copy from Source actions in sequence, or consider Summarize from Source with an instruction to include all relevant content.
