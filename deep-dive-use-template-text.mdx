## What It Does

Inserts text directly from your Word template into the generated document, without referencing any source files. The content comes from the template itself — either the instructional text in the section or the example content from your example document.

## How It Works

When Artos executes this action, it uses the text found in your Word template for the section. This might be boilerplate language, standard regulatory text, or any content that should appear consistently across every document generated from the template.

**No source files involved.** This action doesn't read from your uploaded source files. The content is fixed — it comes from the template.

**Useful for static content.** If a section always contains the same text regardless of the study (a standard disclaimer, a regulatory statement, a fixed methodology description), Use Template Text is the right action.

## Example Instructions

Because the content comes from the template itself, instructions for this action are minimal. You might use instructions to:

- Specify minor variations ("Use the EU version of this text")
- Indicate where the text should be placed relative to other content
- Flag that the text should appear exactly as written with no modifications

## When to Use This Action

**Use Use Template Text when:**

- The section content is always the same regardless of the study
- You're inserting standard regulatory language, disclaimers, or boilerplate
- The content in the Word template is exactly what should appear in the document
- No source data is needed to populate the section

**Use a different action when:**

- The section needs to draw on source files → use **Copy from Source**, **Summarize from Source**, or another source-based action
- The section has placeholders that need to be filled from source content → use **Fill in the Blank**
- The section needs to summarize content from within the document → use **Create Intra-Document Summary**
