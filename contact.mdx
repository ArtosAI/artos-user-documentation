## What to Include When Contacting Support

To help us assist you quickly and effectively, please include:To help us assist you quickly and effectively, please include:

<Tabs>
  <Tab title="Template Questions">
    - Template name and document type
    - Specific section or action you're asking about
    - What you're trying to achieve
    - Example of expected vs. actual behavior (if applicable)
  </Tab>
  <Tab title="Document Generation Issues">
    - Document name and template used
    - Description of the issue
    - Screenshots or examples of unexpected output (if applicable)
    - Any context files used
  </Tab>
  <Tab title="Technical Issues">
    - Error messages or codes (if any)
    - Steps to reproduce the issue
    - Browser or system information (if relevant)
    - File types and sizes involved
  </Tab>
  <Tab title="General Questions">
    - Context about your workflow or use case
    - What you've already tried (if applicable)
    - Timeline or urgency level
  </Tab>
</Tabs>
