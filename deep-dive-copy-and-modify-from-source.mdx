## What It Does

Copies a passage from a source document and applies modifications before inserting it into the section. Unlike Copy from Source, which is fully verbatim, this action allows the content to be adjusted — changing tense, updating terminology, or making other specified alterations — while still preserving the original structure and meaning.

## How It Works

This action reads your source files, identifies the relevant passage, and copies it with the modifications specified in your instructions.

**Chunk behavior.** Like Copy from Source, this action works with one chunk of text at a time (text between two consecutive headings in a source document).

**Modifications are instruction-driven.** The action applies only the modifications you specify. Common modifications include tense changes (past to present), terminology substitutions (drug code to INN), simplification of language, or formatting adjustments.

**Useful for reusing content across documents.** If a passage from a prior document (a protocol, a CSR, an IB) is 90% right but needs minor adjustments for the current document, Copy and Modify lets you get there without starting from scratch.

<Note>
  We recommend using Copy and Modify to lift a specific table or passage from a source document when you want the content preserved but need minor adjustments. If the content requires substantial rewriting, Summarize from Source is usually a better fit.
</Note>

## Example Instructions

| Weak instruction | Strong instruction |
| --- | --- |
| "Copy and modify the text" | "Copy the study design section from the protocol. Change all verbs to past tense. Replace \[drug code\] with \[INN\]." |
| "Use the source but update it" | "Copy the statistical methods from the SAP. Update references to future tense ('will be analyzed') to past tense ('was analyzed'). Replace 'subjects' with 'patients' throughout." |

## Document-Specific Examples

### Clinical Study Report (CSR)

| Section | Instruction |
| --- | --- |
| Study Design | "Copy the study design section from the protocol. Change all verbs to past tense to reflect that the study is completed." |
| Investigational Product | "Copy the investigational product description from the protocol or IB. Replace the drug code with the INN throughout. Change future tense to past tense." |
| Patient Population | "Copy the patient population description from the protocol. Change future tense to past tense. Replace 'will be enrolled' with 'were enrolled'." |

### Investigator's Brochure (IB)

| Section | Instruction |
| --- | --- |
| Mechanism of Action | "Copy the mechanism of action description from the prior IB edition. Update the compound name to reflect the current INN. Revise any outdated references based on the new source documents." |

### Nonclinical Overview (NCO)

| Section | Instruction |
| --- | --- |
| Pharmacology Summary | "Copy the primary pharmacology summary from the nonclinical study reports. Standardize the compound name to the INN throughout. Adjust tense to past tense." |

## When to Use This Action

**Use Copy and Modify from Source when:**

- The source content is largely correct but needs specific, targeted changes
- You need tense changes, terminology substitutions, or minor reformatting
- Preserving the original structure and meaning is important
- The modification scope is limited and can be specified clearly

**Use a different action when:**

- No modifications are needed → use **Copy from Source**
- The content needs substantial rewriting or condensing → use **Summarize from Source**
- You're working from a prior version of the document and need broader updates → use **Copy and Update**

## Tips

- Be specific about what to modify. "Change tense" is less useful than "change all past tense verbs to present tense." The action applies modifications literally based on your instruction.
- If you need to substitute terminology throughout the passage, list the substitutions explicitly: "Replace 'ABC-123' with 'compound X' throughout."
- For tables specifically, Copy and Modify works well when the table structure should be preserved but column labels, headers, or specific values need updating.
