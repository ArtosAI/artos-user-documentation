## Overview

Once your Artos template is configured, you use it to generate documents. What might take days to draft manually can be generated in hours.

## Document Generation Workflow

### Step 1: Name Document & Select Template

1. Create a new document in your workspace
2. Give it a descriptive name
3. Select the Artos template you want to use

### Step 2: Upload Source Files

Source files are the raw data and documents Artos will draw from to populate your document.

**Supported file formats:** .docx, .pdf, .rtf, .xlsx, .csv

| Document Type | Typical Source Files |
| --- | --- |
| Investigator's Brochure | Nonclinical study reports, previous versions of the IB |
| Clinical Study Report | TFLs (Tables, Figures, Listings), SAPs, Protocols and amendments |
| Nonclinical Overview | Individual nonclinical study reports, toxicology data |

<Tip>
  **The Junior Person Test:** Before uploading a file, ask yourself: "If I handed this file to a junior person on my team with no other context, would they understand what it is and what it contains?" If the answer is no, add clearer labeling or documentation within the file.
</Tip>

### Step 3: Add Context Files (Optional)

Context files provide document-specific information about what content should go where.

| Document Type | Context File |
| --- | --- |
| Clinical Study Report | List of Tables |
| Investigator's Brochure | Content Plan |
| Nonclinical Overview | Content Plan |
| Other Nonclinical Summaries | Content Plan |

<Note>
  Context files apply only to the current document generation. If you find yourself using the same context repeatedly, consider adding those instructions directly to your template instead.
</Note>

### Step 4: Add Document-Level Instructions (Optional)

You can add instructions that apply to the entire document or to specific sections. Useful when study-specific information isn't readily available in the source data.

**Document-wide examples:**

- "Write text at an 8th grade reading level"
- "Focus only on the 100mg dose formulation throughout"

**Section-level examples:**

- "Focus only on specific subgroups for this table"
- "Emphasize pediatric data in this section"

### Step 5: Kick Off the Generation Job

1. Review your selections (template, source files, context files)
2. Click to start the generation job
3. Artos begins processing your sources

### What Happens During Generation

Artos reads through all your source files and applies the template's actions and instructions to each section. It extracts relevant information, organizes it according to your template structure, and generates formatted content — including tables, summaries, and other structured elements you've specified.

**Generation time varies based on:**

- Document length and complexity
- Number and size of source files
- Template complexity (number of actions and custom instructions)
- Content composition — plain text processes fastest, followed by tables, then figures and images

### Step 6: Review Your Draft

Once generation is complete, you'll receive an email notification. Your draft document is ready for review in your workspace.
