## Troubleshooting

### Common Issues and Solutions

| Problem | Likely Cause | Solution |
| --- | --- | --- |
| **Wrong content pulled** | Action instructions not specific enough | Add more detail about what content to look for. Specify source location, key terms, or content characteristics. |
| **Output too long** | No length guidance provided | Add length expectations to action instructions or Custom Instructions (ex. "Limit to 2 paragraphs" or "No more than 1 page"). |
| **Output too short** | Missing content requirements | Explicitly list what must be included. Don't assume Artos knows what's important. |
| **Missing information** | Content not specified in instructions | Add the specific information you need to the action instructions. |
| **Wrong table structure** | Table structure defined in Word template | Edit the table in your Word template and re-upload. MRT controls what fills the table; Word template controls structure. |
| **Wrong section order** | Section order defined in Word template | Edit section order in your Word template and re-upload. |
| **Sections not recognized** | Heading styles don't contain "heading" | Update Word template styles to include "heading" in style names. |
| **Tables not formatting correctly** | Table styles don't contain "table" or "caption" | Update Word template table styles. |
| **Content appearing in wrong section** | Action instructions not specific enough about source | Add source-specific guidance (ex. "from the protocol" or "from the pharmacology section of the CSR"). |
| **Repetitive content** | Same action running multiple times | Review action configuration; remove duplicate actions or adjust instructions. |
| **Content toggle issues** | Section set to generate when it should be manual | Check Content toggle setting for the section. |

### Where to Make Changes

| Change needed | Where to make it |
| --- | --- |
| Add/remove/reorder sections | Word template → re-upload |
| Change table structure | Word template → re-upload |
| Change page layout, margins, fonts | Word template → re-upload |
| Change header/footer content | Word template → re-upload |
| Change which action is used | MRT interface |
| Change action instructions | MRT interface |
| Add/modify Custom Instructions | MRT interface |
| Enable/disable parameters | MRT interface |
| Adjust content for this document only | Post-draft editing |
