## Training Options

### Initial Onboarding

When you first start with Artos, we provide onboarding sessions covering:

- **Platform overview** — Understanding the Artos workflow
- **Template basics** — How templates work and how to use existing templates
- **Document generation** — Creating your first documents
- **Post-draft editing** — Refining and perfecting generated documents

### Advanced Training

For teams ready to dive deeper:

- **Template creation and customization** — Building templates from scratch
- **Template editing** — Fine-tuning actions, parameters, and custom instructions
- **Source file optimization** — Preparing files for best results
- **Workflow integration** — Incorporating Artos into your existing processes

### Role-Based Training

- **End users** — Document generation and editing
- **Template administrators** — Template creation and management
- **Technical teams** — Integrations and advanced features

### Dedicated Support Plans

For teams on dedicated support plans:

- Regularly scheduled office hours
- Ongoing training for new team members
- Refresher sessions on specific features
- Best practices reviews and optimization guidance
- Customized documentation tailored to your templates and workflows

## Training Delivery

## Troubleshooting

### Common Issues and Solutions

| Problem | Likely Cause | Solution |
| --- | --- | --- |
| Wrong content pulled | Action instructions not specific enough | Add more detail about what content to look for. Specify source location, key terms, or content characteristics. |
| Output too long | No length guidance provided | Add length expectations to action instructions or Custom Instructions (e.g., "Limit to 2 paragraphs"). |
| Output too short | Missing content requirements | Explicitly list what must be included. |
| Missing information | Content not specified in instructions | Add the specific information you need to the action instructions. |
| Wrong table structure | Table structure defined in Word template | Edit the table in your Word template and re-upload. |
| Wrong section order | Section order defined in Word template | Edit section order in your Word template and re-upload. |
| Sections not recognized | Heading styles don't contain "heading" | Update Word template styles to include "heading" in style names. |
| Tables not formatting correctly | Table styles don't contain "table" or "caption" | Update Word template table styles. |
| Content in wrong section | Action instructions not specific about source | Add source-specific guidance (e.g., "from the protocol"). |
| Repetitive content | Same action running multiple times | Review action configuration; remove duplicates or adjust instructions. |
| Content toggle issues | Section set to generate when it should be manual | Check the Content toggle setting for that section. |

### Where to Make Changes

| Change needed | Where to make it |
| --- | --- |
| Add/remove/reorder sections | Word template → re-upload |
| Change table structure | Word template → re-upload |
| Change page layout, margins, fonts | Word template → re-upload |
| Change header/footer content | Word template → re-upload |
| Change which action is used | Artos template interface |
| Change action instructions | Artos template interface |
| Add/modify Custom Instructions | Artos template interface |
| Enable/disable parameters | Artos template interface |
| Adjust content for this document only | Post-draft editing |

## Getting Help

If you encounter issues not covered here, contact your Artos representative. When reporting an issue, include:

- What you were trying to do
- What you expected to happen
- What actually happened
- Screenshots of the relevant configuration
- Sample source files (if applicable)
